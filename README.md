# 진행 방식

- 일주일 동안 같은 주제로 진행하는 사람과 각자 공부
  - 3개 주제 공부
- 토요일 3시 비대면 모임
  - 3개 중 2개를 무작위 선택해서 면접하듯이 질문
  - 1개는 지난 번에 공부했던 주제 중 아무거나 선정하여 질문
  - 답변이 부족한 경우 다음 주차에 다시 공부
